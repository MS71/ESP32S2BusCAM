# ESP32S2BusCAM

Fetures:
* ESP32 S2/S3 Mini Modul mit und ohne PSRAM
* USB Micro Port
* JTAG Stecker
* ESP Prog Stecker
* Micro SDCard Slot
* 22 Pin Camera Stecker, Camera wird auf das ESP32 Modul geklappt
* Spannungs Regler für Ext Versorgung
* SPI/I2C Bus Stecker
* Support für Stereo Quad Camara
* 4 Pin OLED Display
* IR LEDs
* Debug LEDs
